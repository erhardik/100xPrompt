# 🚀 100xPrompt.md

> Turn weak AI prompts into structured, high-quality prompts automatically.

`100xPrompt.md` is a reusable **prompt optimization system** designed for **ChatGPT Projects**.

Upload it once into a Project and use it anytime you want dramatically better prompts.

It transforms:
- vague prompts
- rough ideas
- incomplete instructions
- messy context

into:
- clear prompts
- structured prompts
- reasoning-optimized prompts
- production-ready prompts

---

# ✨ What It Does

Instead of this:

```txt
write a linkedin post about my startup
```

You get prompts like:

```txt
You are a LinkedIn content strategist helping me announce my startup launch.

Before writing, ask me:
- What the startup does
- Who it serves
- The biggest pain point it solves
- The CTA destination

Once I answer, write a post that:
- Hooks attention in the first 2 lines
- Uses short paragraphs
- Sounds founder-authentic, not corporate
- Includes one sharp insight
- Ends with a clear CTA

Keep it between 150–220 words.

Think carefully before answering.
```

# 📌 How to Use in ChatGPT

## Step 1 — Download the File

Download:

```txt
100xPrompt.md
```

---

## Step 2 — Open ChatGPT

Go to:

```txt
https://chatgpt.com
```

---

## Step 3 — Create a New Project

In the left sidebar:

```txt
Projects → New Project
```

Example project names:
- `100x Prompt System`
- `AI Prompt Optimizer`
- `Prompt Engineering Workspace`

---

## Step 4 — Upload the File

Inside the Project:
- click the upload / paperclip icon
- upload `100xPrompt.md`

Done.

---

## Step 5 — Start a Chat Inside the Project

Now every chat inside that Project can reference the system.

Example:

```txt
Use 100xPrompt.md to optimize this:

write a better email to client
```

or:

```txt
Use 100xPrompt.md on this rough prompt:

I want a viral LinkedIn post about AI
```

# 🔍 Philosophy

Weak prompt:

```txt
help me write better
```

Strong prompt:

```txt
Write a concise LinkedIn post for technical founders explaining why AI agents fail in production.

Use an authoritative but conversational tone.

Include one contrarian insight and end with a CTA encouraging discussion.
```

Better prompts create better outputs.
